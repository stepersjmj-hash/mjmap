⚙️ 설정

opinet.co.kr에서 발급. 주유소는 Opinet API 사용 (프록시 서버 실행 필요)

developers.kakao.com → 플랫폼 키 → REST API 키 복사. 프록시 경유로 호출하므로 도메인 등록 불필요. 네이버 지오코딩 실패 시 카카오 Places로 폴백.

특정 시군을 선택하면 해당 지역 데이터만 가져와서 지도에 표시돼요. 전체 조회 시 수만 건이라 첫 500~1000개만 나옵니다.

📡 경기도 API 서비스명

data.gg.go.kr 마이페이지 → 활용신청내역 → 해당 API → 요청 URL 에서 확인. 예: https://openapi.gg.go.kr/서비스명?KEY=...
