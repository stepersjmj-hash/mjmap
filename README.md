# 🗺️ 나만의 경기도 지도

경기도 공공데이터(OpenAPI)를 활용해 **지역화폐 가맹점 · 문화행사 · 주유소** 정보를 한 지도에서 확인하고 즐겨찾기할 수 있는 개인용 웹앱입니다.

## 주요 기능

- 📍 **내 위치 기반 검색** - GPS로 현재 위치 주변의 데이터를 자동 조회
- 💳 **지역화폐 가맹점** - 경기지역화폐 사용 가능한 매장 표시
- 🎭 **경기도 문화행사** - 현재 진행 중/예정 문화행사 정보
- ⛽ **주유소** - 경기도 내 주유소 위치와 가격 정보
- ⭐ **즐겨찾기** - localStorage로 브라우저에 저장 (서버 없음)
- 🔄 **지도 이동 후 재검색** - 원하는 지역 중심으로 다시 조회
- 📱 **모바일 반응형** - 휴대폰에서도 원활하게 사용 가능

## 1단계: API 키 발급

### 네이버 지도 API Client ID

1. [네이버 클라우드 플랫폼](https://www.ncloud.com) 가입
2. 콘솔 → **Services** → **Maps** 선택
3. **Application 등록** → "Web Dynamic Map" 선택
4. **Web 서비스 URL**에 아래 URL 모두 등록 (중요!)
   - `http://localhost` (로컬 테스트)
   - `http://127.0.0.1`
   - 파일로 직접 열 경우: `file:///` 또는 빈 값 (불가 시 아래 "실행 방법" 참조)
5. 발급된 **Client ID** 복사

### 경기도 공공데이터 API 키

1. [경기도 공공데이터포털](https://data.gg.go.kr) 가입
2. 사용하려는 데이터셋 신청 (최소 3개):
   - **경기지역화폐 가맹점 현황** (`RegionMnyFacltStus`)
   - **경기도 문화행사 현황** (`GGCULTUREVENTSTATUS`)
   - **경기도 주유소 가격정보** (`GgGasStationPriceStus`)
3. 마이페이지에서 **인증키** 확인

> ℹ️ 데이터셋 이름이나 API 서비스명은 시점에 따라 약간 다를 수 있습니다. 만약 데이터가 안 불러와진다면 경기도 포털에서 신청한 API의 **서비스명(엔드포인트)**을 확인하고 `index.html`의 `loadMoneyData`/`loadCultureData`/`loadGasData` 함수 내 서비스명을 수정하세요.

## 2단계: 실행 방법

### 방법 A: 간단 실행 (권장)

1. `index.html` 파일을 더블클릭해서 브라우저에서 열기
2. 초기 화면에서 두 개의 API 키 입력 → **시작하기**
3. 상단 카테고리 칩(💳🎭⛽)을 눌러 원하는 데이터 표시

### 방법 B: 로컬 서버로 실행 (CORS 문제 발생 시)

일부 환경에서 `file://`로 열면 API 호출이 차단될 수 있습니다. 그럴 땐 Python이 설치되어 있다면:

```bash
# 이 폴더에서 실행
python -m http.server 8000
```

그리고 브라우저에서 `http://localhost:8000` 접속.

### 방법 C: 프록시 서버 (API CORS 차단 시)

경기도 API가 CORS 정책으로 직접 호출이 안 되는 경우, 같이 제공된 `proxy.py`를 사용하세요.

```bash
python proxy.py
# → http://localhost:8080에서 프록시 실행됨
```

그 후 `index.html`의 `API_BASE` 상수를 아래와 같이 수정:

```javascript
const API_BASE = 'http://localhost:8080/proxy?url=https://openapi.gg.go.kr';
```

## 문제 해결

### ❌ "네이버 지도 API 로드 실패"
- Client ID가 정확한지 확인
- 네이버 클라우드 콘솔의 **Web 서비스 URL**에 현재 접속하는 URL이 등록되어 있는지 확인
- `file://`로 열 때 실패한다면 방법 B(로컬 서버)로 재시도

### ❌ "CORS 차단? README의 프록시 안내를 확인하세요."
- 방법 C(프록시 서버) 사용

### ❌ 데이터가 0개로 나옴
- 경기도 포털에서 해당 API 사용 승인을 받았는지 확인 (신청 후 승인까지 시간 소요)
- `index.html`의 서비스명(`RegionMnyFacltStus` 등)이 현재 경기도 포털의 서비스명과 일치하는지 확인
- 브라우저 콘솔(F12)에서 상세 에러 확인

### ❌ 마커는 보이는데 주유소만 없음
- 주유소 데이터는 **한국석유공사 오피넷**에서 별도로 제공될 수도 있습니다. 경기도 포털에 없다면 [오피넷 API](https://www.opinet.co.kr)도 고려해보세요.

## 개인정보 관련

- 모든 API 키와 즐겨찾기는 **브라우저 localStorage**에만 저장됩니다.
- 외부 서버로 전송되지 않으며, 다른 사람이 같은 기기의 같은 브라우저를 쓰지 않는 한 안전합니다.
- 브라우저 데이터 삭제 시 설정도 모두 초기화됩니다.
